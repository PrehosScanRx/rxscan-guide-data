# RxScan Guide Data

Public, versioned medication guide catalog used by RxScan Guide.

This repository contains educational, plain-language medication content intended for patients, families and caregivers.

It does not contain the professional clinical content used by RxScan Paramedic.

## Files

- `catalog/guideCatalog.json`: current published RxScan Guide catalog
- `manifest.json`: catalog metadata and current content version

## Important

RxScan Guide is an educational tool and does not replace advice from a physician, pharmacist or other qualified healthcare professional.
